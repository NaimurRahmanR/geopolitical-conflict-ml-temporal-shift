# Data Directory

Raw source datasets are not committed because they may be large, licensed, or versioned by external providers. This directory documents the expected file layout for exact reproduction.

## Expected Raw Files

Place these files in `data/raw/`:

| File | Source | Notes |
| --- | --- | --- |
| `ucdp_prio_acd_v26_1.csv` | UCDP/PRIO Armed Conflict Dataset v26.1 | Conflict-year or country-year conflict information used to define escalation history and target variables. |
| `world_bank_wdi.csv` | World Bank World Development Indicators | Export in wide form with country, indicator, and year columns. |
| `vdem_country_year.csv` | V-Dem country-year dataset | Governance and democracy indicators. |

The code standardises country and year columns conservatively, but reviewers should document any manual harmonisation in `docs/DATA_DICTIONARY.md`.

## Directory Roles

- `raw/`: unmodified source files.
- `interim/`: intermediate merge or audit files.
- `processed/`: final analysis panel generated by `scripts/run_all.py`.

The `.gitkeep` files preserve the directory structure without committing restricted data.
